# CBR Theory Exam Trainer

Practice app for the Dutch driving theory exam (CBR) — English version.

**Exam format (from April 2025):** 50 questions, need 44/50 to pass, 30 minutes.

**Topics covered:** Priority rules, speed limits, traffic signs, alcohol & drugs, overtaking, motorway rules, lights, vehicle documents, parking, cyclists & pedestrians, insight, emergency vehicles, environment.

Live at: https://drivertest.vercel.app
